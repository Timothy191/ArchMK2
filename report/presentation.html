02 / Technical Architecture

Enterprise Monorepo Stack

  • ⚏

    Turborepo & pnpm Workspaces

    Strict dependency catalogs prevent version drift across applications (`apps/portal`, `apps/cms`, `apps/overview`) and shared packages.

  • ⚏

    Supabase with Row-Level Security (RLS)

    All database tables are secured using PostgreSQL policies scoped to employee authorization records and department assignments.

  • ⚏

    Integrated Tools & Cache Tier

    Redis caching handles fast routing, while local services like LocalStack, n8n workflows, and Langfuse trace logging run concurrently.
