Arch-Mk2 Project Wiki

This folder stores a project-scoped wiki for Arch-Mk2.

Structure:

- wiki.config.md — wiki configuration (auto_research settings)
- wiki/ — markdown pages organized by topic (e.g., concepts/, papers/)
- logs/ — human-readable run logs produced by the research loop
- derived/ — structured run outputs (JSON)

Use the `/wiki` commands from the CLI skill to manage, index, and run auto-research for this wiki.
