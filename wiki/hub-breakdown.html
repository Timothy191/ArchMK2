Theme Token Reference

All tokens defined in packages/theme/src/css/variables.css and registered in packages/theme/src/tailwind/preset.ts. Source of truth: macOS Ventura/Sonoma light palette.

Single source of truth — packages/theme/src/tailwind/preset.ts exposes all CSS vars as Tailwind utilities. All apps consume this preset. Never hardcode hex — use var(--...) or the Tailwind semantic class.
Forbidden — raw box-shadow inline styles, raw Tailwind shadow-sm/md/lg/xl, hardcoded #171717/#363636/#fafafa/#898989 (dark palette).
