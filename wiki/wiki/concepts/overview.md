# Overview

This wiki captures research, design notes, and reference material for the Arch-Mk2 project. It is project-scoped and intended to be indexed by the local wiki tooling.

Seed ideas:

- agent memory and RAG patterns
- portal architecture and middleware
- database migration workflow

To add a page and index it into the project's FTS index, use the CLI: `/wiki page <slug> <rel-path> --title "Title" --type concept`.
